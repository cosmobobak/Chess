General
Version Overview of YATS testsets processed under Arena sorted on gained points and solved positions.
Operation
  • Start ChessPartner and load Pro Deo 1.4
  • Click on the misc tab from the engine info window.
  • Press F4.
  • Double click on Version Overview.
Technical
Analyze PGN stores all results in a textfile called YATS.HST which represents the results of this page.

You can easily switch history files by modifying the [History File = YATS.HST] parameter in the REBEL.INI file.
